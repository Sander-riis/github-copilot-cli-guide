CLI Overview

GitHub Copilot CLI runs as a terminal-native AI assistant. It reads your codebase, executes commands, and makes changes — all through natural language.

The agent operates in your working directory with access to tools like grep, view, edit, and bash.

Environment & Configuration

Configuration lives in .github/copilot-instructions.md at the repository root. This file provides project-specific context that persists across sessions.

Custom instructions shape how the agent approaches your codebase — coding style, preferred tools, testing patterns, and architecture conventions.

Best Practices

Start broad, then narrow. Give the agent context about what you're trying to achieve before diving into specific file changes.

Use slash commands like /help and /clear to manage your session.
