Agents Overview

Agents are autonomous workers that Copilot can delegate tasks to. Each agent runs in its own context window with specific tools and capabilities.

Built-in agents handle common tasks: exploring code, running tests, reviewing changes, and providing feedback on plans.

Agents Reference

Five built-in agents ship with Copilot CLI. They're always available and most are invoked automatically through context inference.

Custom agents are defined in .github/agents/ using markdown files with YAML frontmatter.

Skills Overview

Skills are reusable instruction sets that teach Copilot domain-specific knowledge. Unlike agents, skills don't run autonomously — they augment the main agent's capabilities.

Trigger a skill with a keyword or explicitly invoke it. The skill's instructions merge into the agent's context for that task.
