Welcome

This demonstrates the page load splash — the Copilot icon spirals in, pulses gently, then fades away to reveal the content beneath.

It serves as a branded loading moment that gives the page time to initialize syntax highlighting and build the search index.

How It Works

A fixed overlay covers the viewport with the icon centered. CSS animations handle the entrance (scale + rotate), followed by a breathing pulse.

After a short delay, the overlay fades out with an opacity/visibility transition and is then removed from the DOM entirely.
